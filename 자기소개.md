    "레거시 운영부터 최첨단 AI 파이프라인 구축까지, 엔드투엔드 서비스 구현 능력"저는 4년 이상의 기간 동안 Java/Spring 기반의 대규모 레거시 시스템 운영부터, Nest.js/TypeScript를 활용한 현대적 웹 서비스 고도화, 그리고 최근의 LLM 기반 AI 서비스 개발에 이르기까지 폭넓은 기술 스택을 실무에 적용해 왔습니다.AI 엔지니어링 및 RAG 최적화: ArXiv 논문 기반 Hybrid RAG 시스템을 구축하며, 단순 벡터 검색의 한계를 넘어 BM25와 RRF를 결합한 하이브리드 서치를 도입했습니다. 그 결과 Faithfulness와 Context Recall을 각각 10% 향상시켰으며, RAGAS를 통한 정량적 평가 체계를 마련했습니다.멀티 에이전트 협업 자동화: AutoGen을 활용해 기획부터 디버깅까지 무인 자동화되는 2D 게임 생성 파이프라인을 구축했습니다. 특히 LLM의 환각 현상과 API Rate Limit 등 기술적 제약 사항을 FSM(상태 기계) 제어와 커스텀 딜레이 로직으로 해결하며 시스템의 안정성을 확보했습니다.성능 최적화 및 시스템 고도화: 웹 서비스 로딩 속도를 32% 개선하고, EMR 시스템 연동 API 설계 및 대시보드 시각화를 통해 데이터 기반의 의사결정 환경을 구축했습니다. 단순히 기능을 만드는 것에 그치지 않고, 사용자 경험(UX)과 비즈니스 효율성을 동시에 고려합니다."기술적 한계를 돌파하는 집요함과 비즈니스 가치를 창출하는 아키텍처 설계"첫째, 문제의 본질을 파고드는 `해결사`형 개발자입니다.AutoGen 프로젝트 당시 LLM의 Skeleton 코드 출력(AI 게으름) 문제를 해결하기 위해 강력한 프롬프트 엔지니어링과 Playwright 기반의 자동 QA 루프를 도입했습니다. 주어진 도구를 사용하는 데 그치지 않고, 제약 사항을 극복하기 위한 논리적인 대안을 설계하는 데 강점이 있습니다.둘째, 최신 기술을 실무 가치로 전환하는 `빠른 학습력`을 갖췄습니다.로컬 LLM 서비스(Ollama), 멀티모달 AI, CLI 어시스턴트 등 최신 AI 트렌드를 빠르게 습득하여 내부 프로토타입으로 구현해 냈습니다. 새로운 기술 스택(Python, LangChain, Docker 등)을 기존의 탄탄한 백엔드 지식과 결합해 빠르게 실무에 안착시킵니다.셋째, 데이터의 흐름을 이해하는 `풀스택 시야`를 보유하고 있습니다.DB 설계, API 최적화, 프론트엔드 UI/UX 개선, 그리고 인프라(Docker)까지 서비스 전 과정을 경험했습니다. 이 덕분에 AI 기능을 도입할 때도 전체 시스템의 부하와 확장성을 고려한 균형 잡힌 설계를 할 수 있습니다.전통적인 백엔드의 안정성과 최신 AI의 혁신성을 연결하여, 실제 가치를 만드는 서비스를 구축하고 서비스의 항상 내일을 설계하는 개발자가 되겠습니다.
