[사내 API 자동 호출 LLM 에이전트 프로토타입 개발]

· 기간 : 2026.05

· 개요 : 자연어 요청을 사내 백엔드 API 호출로 변환·실행하는 LLM 에이전트 프로토타입 개발. OpenAPI 명세를 RAG로 등록해 적합한 API를 자동 선별하고, 안전장치를 거쳐 실제 호출까지 수행

· 기술 스택 : Python, LangGraph, ChromaDB, sentence-transformers, Redis, pytest, 로컬 LLM(llama-server)

· 구현 :

- 임베딩 벡터 검색(ChromaDB)으로 다수 API 중 요청에 맞는 도구를 동적 선별하는 라우팅 구현
- LangGraph 상태 그래프로 계획→정보보완(되묻기)→승인→실행→검증 워크플로 설계
- 쓰기 작업에 사람 승인(HITL) 게이트 적용 및 API 에러 기반 자가 교정(self-correction) 루프 구현
- 의존성 주입 구조로 테스트/운영 백엔드를 분리하고 pytest 자동 회귀 테스트 구축
- 식별자를 코드가 통제하는 안전 설계 및 변경 이력 감사 로그 적재

· 성과 :

- 자연어 요청 → 실제 백엔드 수정까지 엔드투엔드 동작 검증
- HITL·자가 교정·회로 차단 등 안전장치를 자동 테스트로 검증해 LLM 비결정성 통제
- "위험 식별자는 코드가 통제, LLM은 의도 해석만" 원칙으로 오작동에 의한 잘못된 데이터 변경 차단

· PC 환경 : Windows 11, Core Ultra 7 265K, RAM 64GB, RTX 5070 Ti

· LLM서버 : NVIDIA GB10 DGX Spark 128GB
